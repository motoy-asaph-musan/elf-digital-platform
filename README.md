# ELF Digital Platform

A secure digital learning and examination platform for the
English Language Fraternity (ELF) and BRITE Organisation.

## Purpose
To support:
- Timed online examinations
- Digital seminars and live training
- Assignment submission and marking
- Project-based learning (CBC & IB aligned)

## Tech Stack
- Backend: Node.js (NestJS)
- Frontend: React Native
- Database: PostgreSQL
- Real-time: WebSockets
- Media: Cloudinary / AWS S3
- Streaming: YouTube Live / Mux (Phase 2)

## Key Features
- Server-controlled timed exams
- Auto-submit on timeout
- Role-based access (Admin, School, Teacher, Student)
- International timezone support
- Media uploads (photos & videos)
- Project assessment modules

## Development Phases
- Phase 1: Core exam & submission system
- Phase 2: Live seminars & certificates
- Phase 3: Advanced proctoring & analytics

## Exam Integrity
The platform uses server-authoritative timers and session
management to prevent time manipulation or cheating.
